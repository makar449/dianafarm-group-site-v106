# DIANAFARM GROUP v9.1 Dropdown Fix

## Исправлено
- Верхний пункт меню **Услуги** теперь имеет рабочий выпадающий список.
- Dropdown работает на десктопе по наведению / фокусу и по клику на стрелку.
- На мобильной версии стрелка раскрывает список услуг внутри burger-меню.
- Добавлены прямые ссылки на ключевые услуги: ВНЖ / ПМЖ, регистрация компаний, банки, UAE, Asia, БАДы, косметика, фармацевтический консалтинг, нострификация, недвижимость, авто, паркинги, B2B.
- На странице `services.html` карточки получили якоря `#service-...`, чтобы ссылка из dropdown приводила к конкретной услуге.
- При переходе на конкретную услугу карточка мягко подсвечивается золотым premium-outline.
