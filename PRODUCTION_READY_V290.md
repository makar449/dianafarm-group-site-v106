## Важное ограничение

GitHub Pages запускает только статические HTML/CSS/JS. Поэтому серверные функции `/api`, серверная админка, общие правки на всех устройствах и загрузка файлов через backend требуют Node-хостинг: Render/Railway/VPS. Но визуальные исправления, кнопки «Подробнее», шапка, статические страницы и безопасные v290 animations работают на GitHub Pages.
