DIANAFARM GROUP v218 — Blog/Form/Reviews Admin update

Сделано в этой версии:
1. Блог: основная большая двойная бегущая строка сохранена и растянута шире — от левого края контентной области почти до правой рамки, без заезда на карточку с текстом.
2. Блог: мелкие нижние/серые дублирующие ticker/marquee элементы скрыты.
3. Блог: в видимом блоке вместо “Недвижимость” поставлено “Услуги”.
4. Навигация: добавлены золотые стрелочки перехода между карточками шагов/структуры на desktop.
5. Форма “Оставьте заявку”: улучшены табы Консультация / Недвижимость / Авто, поля ввода, фокус, стеклянная премиальная подложка.
6. Админка: добавлен раздел “Отзывы”. Можно добавлять, редактировать, скрывать и модерировать отзывы через статус pending / approved / rejected.
7. Публичные отзывы: показываются только visible + approved. При подключенном Supabase отзывы могут подтягиваться из таблицы dfg_reviews.
8. Backend: добавлены функции reviews в remote-backend.js и таблица reviews в backend-config.js. SQL находится в SUPABASE_SETUP_V218_REVIEWS.sql.

Важно:
- GitHub Pages сам по себе статический. Локальная админка сохраняет данные в браузере.
- Для общего backend на разных устройствах включите Supabase в assets/js/backend-config.js и выполните SQL из SUPABASE_SETUP_V218_REVIEWS.sql.
