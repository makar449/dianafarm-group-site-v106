# Production launch checklist

- [ ] Проверить все страницы на мобильном: главная, услуги, недвижимость, авто, паркинги, UAE, Asia, блог, о компании, контакты.
- [ ] Отправить тестовую заявку из каждой формы.
- [ ] Проверить CRM-заявки в админке.
- [ ] Подключить webhook заявок во вкладке «Интеграции».
- [ ] Добавить Google Analytics ID.
- [ ] Добавить Google Search Console verification code.
- [ ] Проверить sitemap.xml и robots.txt после публикации.
- [ ] Открыть сайт через `?v=113`, чтобы сбросить старый кэш.
- [ ] После утверждения подключить домен и CMS/backend для общей админки.
